# todo-server

## URL

https://localhost:301/

## Get All ToDo
- URL :

  https://localhost:301/api/getAllToDo

- Method:

  GET

- URL Params

  None

- Data Params

  Required:

  None

## Add ToDo
- URL :

  https://localhost:301/api/addToDo

- Method:

  POST

- URL Params

  None

- Data Params

  Required:

  {
  
      name: "do something",
      
      status: "ongoing"
      
  }
  
  ## Delete ToDo
- URL :

  https://localhost:301/api/deleteToDo

- Method:

  DELETE

- URL Params

  None

- Data Params

  Required:

  {
  
      id: "1",
      
  }
